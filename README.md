# everyday-webtools

A collection of simple, useful, and personal vanilla HTML web projects created to make everyday life easier, more organized, and a little more fun.

This repository contains standalone HTML files for small day-to-day web tools and creative pages, such as a period tracker, Valentine’s Day surprise website, study tracker, portfolio pages, and other personal utility websites.

Each project is built as a simple HTML file, usually with embedded or linked CSS and JavaScript depending on the project.

## Purpose

The goal of this repository is to create practical mini web pages that can help with:

- Personal productivity
- Daily planning
- Study tracking
- Health and wellness tracking
- Creative surprises
- Personal branding
- Portfolio creation
- Small lifestyle utilities

These projects are intentionally simple and lightweight. They are easy to open, edit, customize, and reuse.

## How the Repository Is Organized

All projects are kept as standalone vanilla HTML files in the root folder.

Example:

```text
everyday-webtools/
├── luna-tracker.html
├── valentines.html
├── study-tracker.html
├── portfolio.html
├── habit-tracker.html
└── README.md
```

There is no strict folder structure. Each file represents one independent mini project.

## How to Use

1. Clone this repository:

```bash
git clone https://github.com/your-username/everyday-webtools.git
```

2. Open the repository folder.

3. Open any `.html` file directly in your browser.

4. Customize the page based on your need.

## How to Customize

Because these are simple vanilla HTML projects, you can customize them easily by editing the HTML files directly.

You can change:

- Text content
- Colors
- Images
- Layout
- Buttons
- Animations
- JavaScript logic
- Personal messages
- Form fields
- Tracking rules

No build step, framework, or installation is required.

## Why This Repository Exists

Not every web project needs to be large, complex, or framework-based.

Sometimes a single HTML file is enough to solve a small real-life problem, create a useful tracker, make a personal dashboard, or surprise someone special.

This repository is a collection of those small but useful web projects.
